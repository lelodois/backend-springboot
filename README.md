## Vestibular Backend

Versão backend do sistema de vestibular

* ** Frontend ** - *Angular* - [Source code](https://github.com/lelodois/vestibular-frontend)

### Começando

Clone este projeto e execute o comando abaixo:

```
  mvn install && java -jar target/spring-angular-1.0.0.jar
```

### Pré-requisitos

```
  Java 8
  Maven
```
### Tecnologias

* RestAPI
* Java 8
* Maven
* Spring boot
* Tomcat embedded
* Hibernate
* Swagger
* H2
* Jackson
